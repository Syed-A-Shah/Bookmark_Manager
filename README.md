# Bookmark Manager

Ease for you, minimal effort!
![Bookmark Manager domain model](./public/images/bookmark_manager_1.png)


## How to use

### To set up the project

Clone this repository and then run:

```
bundle
```

### To run the Bookmark Manager app:

```
rackup -p 3000
```

### To run tests:

```
rspec
```

### To run linting:

```
rubocop
```
